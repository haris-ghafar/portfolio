# Portfolio Landing Page

A responsive Portfolio Landing Page built using **HTML5** and **CSS3** as part of an assignment in an **AI Full Stack Engineering Fellowship Program**.

## About the Project

This project was developed as a mentor-assigned task during my AI Full Stack Engineering Fellowship. The goal of the assignment was to strengthen core front-end development skills by creating a multi-section static website while applying fundamental CSS layout techniques and semantic HTML practices.

The portfolio website showcases a modern and responsive design featuring sections such as Hero, About, Skills, Projects, Contact, and Footer.

## Learning Objectives

The project focuses on the following concepts:

* Semantic HTML
* CSS Box Model
* Margin and Padding
* Relative and Absolute Positioning
* Flexbox Layout
* CSS Grid Layout
* Responsive Web Design
* Clean UI Structure

## Features

### Navbar

* Built using Flexbox
* Responsive navigation links
* Clean and modern layout

### Hero Section

* Professional introduction area
* Demonstrates relative and absolute positioning
* Includes a status badge (e.g., Open to Work)

### About Section

* Two-column layout
* Demonstrates box model concepts including margin, padding, and borders

### Skills Section

* Skill badges displayed using Flexbox
* Responsive wrapping for different screen sizes

### Projects Section

* Responsive CSS Grid layout
* Project cards with descriptions and links

### Contact Section

* Contact information and social links
* Flexbox alignment

### Footer

* Simple centered footer using Flexbox

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
